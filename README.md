## Installation
Clone this repo 
```shell
git clone https://github.com/brmatthy/.dotfiles.git ~/.dotfiles
```

Init the submodules
```sh
git submodule update --init --recursive
```

### Install the dotfiles for the programs
```shell
./dotfiles.sh
```
