# dotfiles

## Install distro
Do a minimal install of the distro of choice.
make sure to install the following packages and software
#### packages
- xorg
- i3
- polybar
- rofi
- networkmanager
- pulseaudio
- ligthdm
- ligthdm-gtk-greeter
- git
- neofetch

#### programs
- firefox
- bitwarden
- spotify
- discord

## Install dotfiles
clone this repo
```sh
git clone git@github.com:brmatthy/dotfiles.git
```
emplace the dotfiles
```sh
./setup.sh
```
